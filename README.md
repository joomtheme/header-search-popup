## Features

- Menu icon trigger
- Automatic popup positioning
- Responsive behaviour
- ESC key to close
- Click outside to close
- Configurable popup width
- Mobile breakpoint support
- English and Turkish language support
- Compatible with Joomla 6

## Installation

1. Download the module ZIP file.
2. Go to **Joomla Administrator → System → Extensions → Install**.
3. Upload the ZIP file.
4. Publish the module.

## Menu Item Setup

Create a menu item with the following settings:

- **Menu Item Type:** System Links → URL  
- **Link:** `#`  
- **Display Menu Item Title:** No  
- **Link Class:** `js-search-toggle`  
- **Link Icon Class:** `fa fa-search`

This menu item will trigger the search popup.

## Module Settings

Recommended settings:

- Alignment mode: Auto
- Popup width: 300px
- Mobile breakpoint: 768px
- Viewport gap: 10px
